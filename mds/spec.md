# YkMake 프로젝트 30일 개발 계획 (v2 - 커리큘럼 기반)

## 개요

이 문서는 제공된 "Maker 마스터클래스" 커리큘럼과 `wemake.cool` 데모 사이트를 기반으로 재구성된 30일 개발 계획입니다. 각 날짜별 목표는 커리큘럼 순서를 최대한 따르면서 실제 개발 흐름에 맞게 조정되었습니다.

## 참고 자료

-   **커리큘럼:** 사용자 제공 이미지
-   **데모 사이트:** [https://wemake.cool/](https://wemake.cool/)
